#Machine Learning Project# Summary Analysis of exercise monitoring variables to predict how correct movements are (class A to E). Several models are tested with different levels of data pre-processing.

Objective Identify the model that better predicts the test data using the training data.

##Data loading and cleaning## Variables with close to zero variance, with more than .95 ratio of a single outcome or with an identifier are removed from the data sets.

##Preprocessing, models y predictions## When pre-processing is performed data is centered, scaled and principal component variable merging is applied. Else models are applied directly to data. The glm model can not be applied since it is for outcomes with only two classes, neither do nb models work. Models are run with lda and rpart methods.
